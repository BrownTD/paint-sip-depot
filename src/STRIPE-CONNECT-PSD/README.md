# Stripe Connect – Paint & Sip Depot

This module handles Stripe Connect embedded onboarding and dashboards
for Paint & Sip Depot event hosts.

## What this supports

- Express account creation
- Embedded onboarding
- Compliance & notifications
- Payments & payouts dashboards
- Stripe webhooks

## Key folders

- `server/` – Stripe business logic
- `api/` – Next.js route handlers
- `client/` – React Connect components
